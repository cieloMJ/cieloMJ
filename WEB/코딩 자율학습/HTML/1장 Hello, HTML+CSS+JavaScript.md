# 1. 개발 환경 설정하기
## 1) 코드 에디터 설치
- 실무 : 아톰(Atom), 서브라임 텍스트(Sublime Text), 브라켓(Brackets), 비주얼 스튜디오 코드(Visual Studio Code)와 같은 코드 에디터 사용
- -> VS Code 설치

## 2) VSCode 확장 프로그램 설치
- 확장 프로그램(EXtension) : 프로그램명(Extension Name)/제작자명(Publisher Name)
  - 확장 프로그램명으로 검색이 안 되는 경우 제작자명으로 검색
 
- Live Server / Ritwick Dey
  - 가상 서버를 이용해, 작성한 HTML 코드를 실시간으로 확인할 수 잇는 확장 프로그램
- Auto Rename Tag / Jun Han
  - 시작 태그와 종료 태그 둘 중 한 쪽을 수정했을 때 다른 쪽을 자동으로 수정하도록 설정
- HTML to CSS au9tocompletion / solnurkarim
  - HTML 문서에서 class 속성으로 작성한 값을 연결된 CSS 파일에서 자동 완성하도록 지원
- HTML CSS Support / ecmel
  - CSS 파일에서 작성한 클래스나 아이디 선택자 값을 연결된 HTML 문서의 id 속성과 class 속성값으로 적을 때 자동완성 지원
- Code Runner / Jun Han
  - 자바스크립트 코드 실행ㅇ르 VSCode 내부에서 하도록 지원
- Korean Language Pack for Visual Studio Code / Microsoft
  - 메뉴와 설명을 한글로 변경
  - 온라인에서 VSCode 관련 검색 시 나오는 자료들은 대부분 영문이라 매칭하기 어려울 수 있음
 
## 3) 웹 브라우저 설치 
- VSCode로 작성한 HTML 코드를 실행하려면 웹 브라우저 필요
- 기본으로 설치된 엣지(마이크로 소프트), 사파리(애플) 외에도 크롬 웹 브라우저 사용
