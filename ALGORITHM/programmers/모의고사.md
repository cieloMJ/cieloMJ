### 풀이
```
def solution(answers):
    
    p1 = [1, 2, 3, 4, 5]
    p2 = [2, 1, 2, 3, 2, 4, 2, 5]
    p3 = [3, 3, 1, 1, 2, 2, 4, 4, 5, 5]

    patterns = [p1, p2, p3]

    # 정답 개수
    score = [0, 0, 0]

    # 비교
    for i, ans in enumerate(answers):
        for j, pattern in enumerate(patterns):
            if ans == pattern[i%len(pattern)]:
                score[j] += 1
    max_score = max(score)
    result = [i+1 for i, score in enumerate(score) if score == max_score]
    return result
```

1, 2, 3번 사람이 찍는 것의 루틴이 있는 것은 쉽게 파악했지만
이를 정답과 비교하는 방법에 많은 생각을 한 문제였다.

처음에는 enumerate를 생각하지 못했기 때문에 딕셔너리로 각 값이 위치한 인덱스를 추출한 뒤
두 개의 인덱스 리스트가 동일한 것의 갯수를 세는 방식으로 진행하려고 했다.

그렇게 하는 것도 하나의 방법이지만, 값을 찾고 이후에 비교하고 하는 방식이 비효율적이라고 생각했다.

함께 스터디 하는 사람이 리스트는 enumerate 많이 사용한다는 것이 생각나
딕셔너리가 아닌 리스트로 접근하여 풀었다.
