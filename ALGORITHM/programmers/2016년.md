## 문제 풀이

```
from datetime import datetime

def solution(a, b):
    day = ['MON', 'TUE', 'WED', 'THU', 'FRI', 'SAT', 'SUN']
    year = 2016    
    date = datetime(year, a, b)
    week_idx = date.weekday()
    return day[week_idx]
    ```

기존에 생각한 풀이 방법은 월별로 7로 나눈 뒤 나머지를 통해서 요일을 추측하는 방법이었다.
해당 방법의 경우 12달의 시작 요일을 알고 있거나 29, 30, 31 중 어떤 값을 가지고 있는지 우선 파악하는 것이 필요했다.
그렇기 때문에 위의 풀이보다 더 긴 수식이 필요했고, 직관적일 수 있지만 12개를 모두 신경 써야 하기 때문에
오류가 발생할 경우도 있었다.

그렇기에 datetime를 import한 뒤,
해당 연, 월, 일이 어떤 요일인지를 index로 접근했다.
weekday()의 경우 월요일이 0, 일요일이 6의 값을 가지기 때문에
순서를 월요일부터 일요일까지로 배치한 뒤 요일을 return했다.
